About this app
This is Searchos web app built by Eze, a fullstack web developer during the global Covid19 pandemic in 2020 to help in locating health care centers in emergency cases. It is easy to use: just input a number on the text box representing how many meters away you want to search and click the search button. Have a nice time using this feature
